# Monty Examples

Numerous examples of what monty can do, and how.

## Included Examples

1. `examples/datetime_planning` - Date/time calculations (`today`, `+1 hour`, next Thursday at 3pm, first Monday next month).
2. `examples/expense_analysis` - External async function orchestration and typed tool calling.
3. `examples/sql_playground` - Mixed CSV/JSON analysis with sandboxed file access and external functions.
